Belajar table

untuk membuat table di html kita bisa memakai tag table dalam setelan default table tidak memiliki border, maka harus kita tambahkan dengan style css

ada beberapa tag yg terdapatpada table thead untuk mebuat header pada table
tbody untuk membuagrup isi pada table
tfoot untuk membuat group footer pada table
th untuk membuat kolom pada header tr untuk membuat baris pada table
td untuk membuat kolom pada table



Colgroup

kolom grup dilakukan untuk membuat grup dari beberapa kolom, biasanya digunakan untuk memberi style pada kolom yang kita inginkan saja saja akan capek jika kita melakukannya satu satu pada tiap kolom maka dari itu kita menggabungkannya ke kolom grup

Kita bisa memakai tag colgroup untuk membuat grup kolom dan tag col untuk tiap kolom dan jika kita ingin menambahkan style terhadap lebih dari satu kolom dengan style yang sama kita bisa memakai tag span disertai dengan sebuah nilai contohya : span= "2", maka berarti kolom pertama sampai angka yang tertulis pada nilai span, pada contoh nilai span berisi angka 2 maka berartibkolom 1 sampai kolom ke 2, jika nilaispan adalah 4 maka kolom satu sampai kolom ke empat, dan jika membuat tag span lagi maka kolom akan berlanjut pada hitungan kolom berikutnya contoh tag span kedua bertulis nilai 6 maka berarti kolom ke 5 sampai ke 6 karena nilai span yang bertama adalah 4

Contoh Hasil :

Nama Alamat No Handphone
Anji Palembang 0211111
nisa lubuk linggau 0212222
Jodi way kanan 0213333

pada contoh diatas saya memakai nilai span 2 pada colgroup maka kolom 1 dan dua akan memakai style yang sama yaitu warna biru dengan cukuk memberi style pada colgroup tak perlu satu-satu pada kolom satu dan dua
