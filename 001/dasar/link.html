Membuat Link

dalam membuat web, biasanya kita akan membuat banyak sekali halaman html disinilah fungsi dari link untuk berpindah dari halaman satu ke halaman lainnya tanpa manual dengan cara mengetikannya di search web browser
dalam membuat link di html kita biasanya memakai tag a yang mana pada tag a memiliki beberapa atribute di didalamnya
  1. Atribute href
    2. atribute href berisi lokasi tujuan dari link yang akan dituju
  2. atribute target
    3. atribute target memiliki fungsi sebagai target dari jendela web browser jika self artinya halaman akan ditampilkan pada halaman yang sama dan ini adalah setelan default jadi, jika kita tidak menyeting target maka ia akan muncul di halam yang sama
    selanjutnya ada blank yang mana jika kita memilih blamk maka halaman yang ingin kita tuju akan terbuka di jendela baru di browseryang dipakai
  3. atribute Title
    4. seperti makna tertulis, atribute title untuk menuliskan judul yang akan keluar ketika mouse berada diatas link tersebut
