Entities

Di dalam HTML, beberapa karakter sudah dipesan atau dikunci oleh HTML, sehingga kita tidak bisa menggunakan karakter seperti **`<`** atau **`>`** pada tulisan teks biasa. Jika kita memaksakan untuk memasukannya, halaman HTML akan rusak dan error karena karakter tersebut akan dianggap sebagai tag HTML.

Break Line

Penulisan di dalam paragraph pada HTML, fitur *enter* tidak akan dianggap *enter* oleh HTML itu sendiri. Jika ingin membuat baris baru, disarankan memakai paragraf yang terpisah dari paragraf awal. Namun, jika kita ingin memakai *enter* di tengah paragraf, kita bisa memakai tag **`<br>`**. Tag ini adalah *void element* sehingga tidak memerlukan konten.

Formatting

Di dalam aplikasi pengolahan teks seperti MS Word, kita bisa memformat tulisan menjadi tebal atau miring. Begitu juga dengan HTML, kita bisa merubah format teks menggunakan tag yang sesuai dengan yang kita inginkan:

  • **`<strong>`** untuk membuat teks **penting** (sebagai pengganti `<b>`)
  • **`<em>`** untuk memberi *penekanan* pada teks (sebagai pengganti `<i>`)
  • **`<small>`** untuk teks kecil
  • **`<del>`** untuk mencoret teks
  • **`<ins>`** untuk meberi garis bawah pada teks
  • **`<sub>`** untuk subscript teks
  • **`<sup>`** untuk superscript teks

Membuat Link

Dalam membuat web, biasanya kita akan membuat banyak sekali halaman HTML. Di sinilah fungsi dari *link* untuk berpindah dari halaman satu ke halaman lainnya. Dalam membuat *link* di HTML kita memakai tag **`<a>`** yang mana memiliki beberapa atribut:

  1. Atribut **`href`**: Berisi lokasi tujuan dari link yang akan dituju.
  2. Atribut **`target`**: Memiliki fungsi sebagai target dari jendela web browser. Jika **`_self`** artinya halaman akan ditampilkan pada halaman yang sama (ini adalah setelan *default*). Selanjutnya ada **`_blank`** yang mana halaman yang ingin kita tuju akan terbuka di jendela baru di *browser*.
  3. Atribut **`title`**: Untuk menuliskan judul yang akan keluar ketika *mouse* berada di atas *link* tersebut.

Memasukkan Gambar

Tag **`<img>`** digunakan untuk memasukkan gambar atau foto ke dalam web kita untuk memperindah tampilan yang akan kita buat. Hampir semua format yang didukung oleh web browser bisa kita tampilkan.

Attribute `<img>`

  1. Atribut **`src`**: Untuk menentukan *source* atau asal gambar, bisa menggunakan *absolute url* atau *relative url*.
  2. Atribut **`alt`**: Digunakan untuk representasi teks atau tulisan bagi gambar (penting untuk aksesibilitas).

Favicon

Ketika kita membuka *website* seperti Google, Facebook, dan lainnya, biasanya pada bagian tab *web browser* terdapat logo dari perusahaannya.

Untuk membuat *favicon*, kita bisa memakai tag **`<link>`** pada bagian **`<head>`** pada HTML, di mana kita perlu menggunakan atribut **`rel="icon"`** dan **`href`** menuju lokasi gambar *icon* yang ingin dipakai. Gambar *icon* bisa berupa `.ico`, `.png`, `.jpg`, `.svg`, atau `.gif`.

Memasukkan Video

Selain gambar, pada HTML juga bisa memunculkan video di halaman web yang kita buat. Biasanya format video yang digunakan adalah **`.mp4`**, **`.webm`** atau **`.ogg`**. Untuk menampilkan video, kita memakai tag **`<video>`**.

Tag **`<video>`** memiliki beberapa atribut penting:

  • **`width`** dan **`height`**: Untuk ukuran lebar dan tinggi video.
  • **`controls`**: Untuk menampilkan kontrol pemutar video.
  • **`autoplay`**: Untuk membuat video otomatis terputar.
  • **`<source>`**: Berisi konten asal video yang ingin ditampilkan.

Audio

HTML juga mendukung untuk memasukkan audio memakai tag **`<audio>`**. Penggunaannya mirip seperti penggunaan tag video.

Inline-Frame

Inline Frame digunakan jika kita ingin menambahkan halaman web lain pada halaman web kita. Untuk melakukannya, kita bisa memakai tag **`<iframe>`** pada HTML.

Belajar Table

Untuk membuat tabel di HTML kita bisa memakai tag **`<table>`**. Dalam setelan *default*, tabel tidak memiliki *border*, maka harus kita tambahkan dengan *style* CSS.

Ada beberapa tag yang terdapat pada tabel:

  • **`<thead>`** untuk membuat *header* pada tabel.
  • **`<tbody>`** untuk mengelompokkan isi pada tabel.
  • **`<tfoot>`** untuk membuat *footer* pada tabel.
  • **`<th>`** untuk membuat kolom *header*.
  • **`<tr>`** untuk membuat baris pada tabel.
  • **`<td>`** untuk membuat kolom data pada tabel.

Colgroup

Kolom grup dilakukan untuk membuat grup dari beberapa kolom, biasanya digunakan untuk memberi *style* pada kolom yang kita inginkan saja. Kita bisa memakai tag **`<colgroup>`** untuk membuat grup kolom dan tag **`<col>`** untuk tiap kolom. Untuk lebih dari satu kolom, kita bisa memakai atribut **`span`** disertai dengan nilai angka (contoh: `span="2"` berarti kolom 1 dan 2).

Colspan dan Rowspan

Kedua tag di atas digunakan untuk menggabungkan beberapa kolom maupun baris (*merge*). Tag **`colspan`** untuk menggabungkan beberapa kolom, sedangkan tag **`rowspan`** digunakan untuk menggabungkan beberapa baris tabel menjadi satu.

Seperti yang terlihat, saya menggabungkan kolom *header* tabel memakai tag **`colspan`** sehingga pada *header* yang awalnya ada dua menjadi satu kesatuan:
